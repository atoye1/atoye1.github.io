## [사이트 링크](https://learngitbranching.js.org/?locale=ko)

- level mixed 1 까지 진행하였으나, interactive rebase, cherrypick 개념이 잘 안서서 어려움을 느낀다.

- git rebase -i
- git cherry-pick <commit name>
- git checkout : HEAD를 바꾸는 것
- git reset : commit을 되돌리는 것 (파일 내용까지 변경)
- git revert : commit을 되돌리되 예전 커밋은 변경하지 않는다.  
	안전하게 사용하기 위해서 git revert후에 git branch 하면 될거같다.
